# PraktikumPBOA2

Repository ini berisi latihan dan tugas praktikum **Pemrograman Berorientasi Objek (PBO)** menggunakan Java.

## Struktur Folder

- **IntroTest/**
  - Contoh program dasar Java (`App.java`).
- **Titik/**
  - Implementasi class `Titik` dan driver `MTitik`.

## Cara Compile dan Run (contoh folder Titik)

Masuk ke folder `Titik`, lalu jalankan:

```bash
javac -d bin .\src\*.java
java -cp bin MTitik
```

## Catatan

- Setiap folder praktikum punya source code di folder `src`.
- File hasil compile disimpan di folder `bin`.

---

Dibuat untuk kebutuhan praktikum PBO.
